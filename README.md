It all starts with a Telegram bot. 
From there, a link leads to the fundraising organizer. 
Next, the fund organizer allows you to create a collection. 
The collection includes a purchase table: it contains a list of items, their quantities, and the total amount. 
There is a separate user table that tracks roles (admin, regular user) and how much each person contributed and spent. 
Once the expenses are tallied, the bot compares them with the contributions. 
If there is a surplus, it is distributed among everyone, and they are notified of how much will be refunded. 
If there is a shortfall, the bot reminds them how much more needs to be contributed. 
The result is a logical flowchart where everything is interconnected.
